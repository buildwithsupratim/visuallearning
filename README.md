# 🌈 MARIA — Visual Learning AI for Kids

MARIA is an AI-powered learning assistant for children that explains lessons through colourful illustrations, image-based storytelling, and visual examples.

Instead of only giving text answers, MARIA generates educational images to help students understand concepts by seeing them. Whether it is science, math, grammar, animals, space, or history, MARIA creates bright and child-friendly visuals that make learning easier, more fun, and more memorable.

## ✨ Features

- 🎨 Generates visual explanations for lessons and questions
- 🖼️ Uses colourful illustrations to teach concepts
- 📚 Supports curriculum-based learning
- 👩‍🏫 Multiple teacher personas for different learning styles
- 🧠 Uses RAG-based knowledge retrieval for better answers
- 🤖 Powered by Qwen3 and SDXL-Turbo
- 🌈 Designed for children aged 6–12
- 💬 Interactive chat-based learning experience

## 🖼️ Why Visual Learning?

Children often understand ideas faster when they can see them. MARIA turns lessons into vivid scenes, colourful illustrations, and simple visual explanations so students can picture what they are learning.

This helps improve:
- Better understanding
- Stronger memory retention
- Higher engagement
- More curiosity and creativity
- Easier learning for younger students

## 📦 Tech Stack

- Qwen3-4B
- SDXL-Turbo
- Gradio
- FAISS
- Sentence Transformers
- Diffusers
- Hugging Face Transformers

## 🎯 Goal

MARIA aims to make learning feel magical by turning every lesson into something children can see, imagine, and enjoy.
